# Health Cost Prediction using Linear Regression (Python)

**Overview:**  
Machine learning project that predicts individual healthcare costs based on demographic and lifestyle factors. Demonstrates data preprocessing, feature engineering, and regression modeling.

**Technologies Used:**  
- Python 3.x  
- Pandas & NumPy  
- scikit-learn / TensorFlow  
- Matplotlib for visualization  

**Features:**  
- Data cleaning and preprocessing  
- Encoding of categorical variables  
- Train-test split for model evaluation  
- Linear Regression model for prediction  
- Evaluation using Mean Absolute Error (MAE)  

**Project Workflow:**  
1. Load and preprocess dataset  
2. Encode categorical features  
3. Split data into training and testing sets  
4. Train Linear Regression model  
5. Evaluate model performance and visualize predictions  

**Key Skills Demonstrated:**  
- Regression modeling and evaluation  
- Data preprocessing and feature engineering  
- Model performance analysis (MAE)  
- Python-based ML workflows
